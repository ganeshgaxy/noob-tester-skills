# noob-analyze

Deep analysis of a ticket — gap, requirements, feasibility, and impact analysis against the codebase. Runs before development starts.

## Installation

```bash
claude plugin add /path/to/noob-tester-skills/plugins/noob-analyze
```

## Usage

```
/noob-analyze
```

Produces 4 analyses: gap, requirements, feasibility, and impact — all derived from the existing codebase before any development begins.
