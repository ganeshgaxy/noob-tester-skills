# noob-report

Generate comprehensive test report — analyses, plans, test cases, execution results, issues, RCA, accessibility. Determine verdict, notify Slack, update ticket.

## Installation

```bash
claude plugin add /path/to/noob-tester-skills/plugins/noob-report
```

## Usage

```
/noob-report
```

Pulls all data for a ticket and produces a full report with verdict (PASS/FAIL/PARTIAL), issue breakdown, execution results, RCA, accessibility findings, and recommendations.
