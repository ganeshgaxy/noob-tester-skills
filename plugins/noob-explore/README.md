# noob-explore

Execute UI test cases via browser automation. Uses run packs for tracking, UI maps for learning, capture-page for evidence. One test case per invocation.

## Installation

```bash
claude plugin add /path/to/noob-tester-skills/plugins/noob-explore
```

## Usage

```
/noob-explore
```

Executes one UI test case per invocation via browser automation — claims a test case, logs in, navigates, captures evidence (screenshots, snapshots, console, HAR), and records results. Invoke repeatedly for all cases.
